Introdução ao Client-Side e Front-End

Client-Side

O Client-Side vamos usar a tradução em português para melhorar sua compreensão o lado do cliente, é a parte da qual o usuário tem acesso através do navegador, acessando o navegador usando um smartphone ou desktop é tudo aquilo que o usuário visualiza, esse site que é composto apenas por um arquivo em HTML e por esse motivo, esse site não é nada parecido com os sites que você, usuário, está acostumado nos dias de hoje com cores, imagens, fundos animados, a página interagindo com o usuário, todas esssas funções anteriormente citadas, são realizadas pelo CSS e JavaScript, sendo assim um site é composto por 3 documentos:

  1. HTML
  2. CSS
  3. JAVASCRIPT

HTML

Finalidade

O HTML é a linguagem de marcação de hipertexto que serve como base para a criação de sites e páginas. É usado para criar a estrutura básica da página, como a criação de títulos, parágrafos, listas, links, imagens e muito mais. Tudo o que é apresentado na página é definido em código HTML, sendo um dos principais componentes de qualquer site e é necessário para a criação de qualquer página web, ajudando a definir o conteúdo da página e a estruturar a informação, criando vários componentes diferentes.

CSS

Finalidade

O Cascading Style Sheets CSS (Folhas de Estilo em Cascata) é uma linguagem de marcação que desempenha um papel essencial na criação de sites modernos e funcionais, sendo responsável por dar ao site sua aparência visual, desde o layout, cores e tipografia até o design detalhado de botões, menus e outros elementos.
Além disso, também é responsável por tornar os sites acessíveis para todos, pois permite que os usuários ajustem as cores e fontes para melhorar a experiência de usuário O CSS ajuda a tornar os sites mais atraentes, intuitivos e fáceis de navegar.

JAVASCRIPT

Finalidade

JavaScript (JS) é uma linguagem de programação de alto nível que é usada principalmente para criar páginas web e aplicações web. Tendo por objetivo adicionar interatividade, animações, efeitos visuais e também para criar interatividade entre o usuário e o site. É usado para tornar as páginas web dinâmicas, permitindo que os usuários façam alterações sem precisar atualizar toda a página. Além disso, pode ser usado para criar aplicações web baseadas em eventos, que permitem que as ações dos usuários sejam detectadas e respondidas. JavaScript é usado em praticamente todos os sites da internet, pois oferece inúmeras possibilidades de interatividade.

Estrutura básica do HTML

É composta por duas tags, head e body

Head

É usada para conter elementos que fornecem informações sobre o documento HTML, esses elementos são usados para identificar o documento, definir o estilo, carregar scripts, definir meta tags e definir links para outros documentos. Os elementos dentro da tag head são geralmente invisíveis para os usuários finais, mas são fundamentais para que o navegador receba informações importantes sobre o site e o funcionamento correto do documento os elementos dentro da tag head incluem título, meta, link, style, script, entre outros. A tag head deve ser a primeira tag de todos os documentos HTML e deve conter todos os elementos necessários para o documento.

Body

A tag body é generica no HTML que delimita o corpo principal de um documento HTML, é o contêiner principal de todos os outros elementos que exibem informações e conteúdos, tais como textos, imagens, links, formulários, etc. Esses elementos são inseridos dentro da tag body, para que possam ser exibidos no navegador sendo muito importante para a criação de documentos HTML, já que é o contêiner principal para o conteúdo do documento.

Textos

O texto é formatado e organizado utilizando tags HTML, que são instruções para o navegador exibir o texto de uma maneira específica, algumas das tags de texto mais comuns em HTML incluem as tags de cabeçalho (h1, h2, h3, etc.) que são usadas para títulos e sub-títulos, a tag de parágrafo (p), que é usada para organizar texto em blocos de parágrafos, e a tag de quebra de linha (br), que é usada para criar uma nova linha de texto.

Listas e Links

Listas e links são elementos importantes do HTML, que permitem aos desenvolvedores criar conteúdo interativo e organizado em páginas da web. As listas são criadas usando tags HTML como ul (lista não ordenada) e ol (lista ordenada) e permitem que o conteúdo seja exibido em um formato organizado e fácil de ler. Essas tags podem ser combinadas com a tag li (item da lista) para criar uma lista de itens com marcadores, números ou outras formas de organização.
Já os links são usados para conectar conteúdo em diferentes partes de uma página da web ou em outras páginas da web. Eles são criados usando a tag a (anchor) e permitem que o usuário clique em um texto ou imagem e seja direcionado para outra página ou seção da página atual. Além disso, os links também podem incluir atributos adicionais, como o atributo "target" que controla como o link é aberto e "rel" que define o relacionamento do link com a página vinculada. Em resumo, as listas e os links são elementos importantes do HTML que permitem que o conteúdo seja organizado e acessível para o usuário final.
